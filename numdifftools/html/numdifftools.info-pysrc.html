 1  """ 
 2  NUMDIFFTOOLS 
 3  ============ 
 4  Suite of tools to solve automatic numerical differentiation 
 5  problems in one or more variables. All of these methods also 
 6  produce error estimates on the result. 
 7  A pdf file is also provided to explain the theory behind these tools. 
 8   
 9  To test if the toolbox is working paste the following in an interactive 
10  python session:: 
11   
12     import numdifftools as nd 
13     nd.test(coverage=True) 
14   
15  Derivative: 
16  ----------- 
17  A flexible tool for the computation of derivatives of order 1 through 4 
18  on any scalar function. Finite differences are used in an adaptive manner, 
19  coupled with a Romberg extrapolation methodology to provide a maximally 
20  accurate result. The user can configure many of the options, changing 
21  the order of the method or the extrapolation, even allowing the user to 
22  specify whether central, forward or backward differences are used. 
23   
24  Gradient 
25  -------- 
26  Computes the gradient vector of a scalar function of one or more variables 
27  at any location. 
28   
29  Jacobian 
30  -------- 
31  Computes the Jacobian matrix of a vector (or array) valued function of 
32  one or more variables. 
33   
34  Hessian 
35  ------- 
36  Computes the Hessian matrix of all 2nd partial derivatives of a scalar 
37  function of one or more variables. 
38   
39  Hessdiag 
40  -------- 
41  The diagonal elements of the Hessian matrix are the pure second order 
42  partial derivatives. 
43   
44  Examples 
45  -------- 
46  Compute 1'st and 2'nd derivative of exp(x), at x == 1:: 
47   
48      >>> import numpy as np 
49      >>> import numdifftools as nd 
50      >>> fd = nd.Derivative(np.exp)              # 1'st derivative 
51      >>> fdd = nd.Derivative(np.exp, n=2)  # 2'nd derivative 
52      >>> fd(1) 
53      array([ 2.71828183]) 
54   
55  Nonlinear least squares:: 
56   
57      >>> xdata = np.reshape(np.arange(0,1,0.1),(-1,1)) 
58      >>> ydata = 1+2*np.exp(0.75*xdata) 
59      >>> fun = lambda c: (c[0]+c[1]*np.exp(c[2]*xdata) - ydata)**2 
60      >>> Jfun = nd.Jacobian(fun) 
61      >>> np.abs(Jfun([1,2,0.75])) < 1e-14 # should be numerically zero 
62      array([[ True,  True,  True], 
63             [ True,  True,  True], 
64             [ True,  True,  True], 
65             [ True,  True,  True], 
66             [ True,  True,  True], 
67             [ True,  True,  True], 
68             [ True,  True,  True], 
69             [ True,  True,  True], 
70             [ True,  True,  True], 
71             [ True,  True,  True]], dtype=bool) 
72   
73  Compute gradient of sum(x**2):: 
74   
75      >>> fun = lambda x: np.sum(x**2) 
76      >>> dfun = nd.Gradient(fun) 
77      >>> dfun([1,2,3]) 
78      array([ 2.,  4.,  6.]) 
79   
80   
81  See also 
82  -------- 
83  scipy.misc.derivative 
84
